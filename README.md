# Gestión de Estudiantes - Java POO

Proyecto desarrollado en Java aplicando los conceptos de:

- Clases
- Objetos
- Constructores
- Métodos
- Encapsulación
- Paquetes (model y app)

El programa permite crear estudiantes y verificar si aprobaron según su promedio.
